Vehicle Driver Management System

Project Description

The Vehicle Driver Management System is a backend application built using the MERN stack (Node.js, Express.js, and MongoDB). This system allows users to register and log in securely using JWT authentication. After authentication, users can manage driver details such as adding new drivers, viewing the list of drivers, updating driver information, and deleting drivers.

The application also supports image upload for driver photos and implements a soft delete mechanism to track deleted drivers.

This project was built to practice backend development concepts such as REST APIs, authentication, database operations, and MVC architecture.
